# Customer Behavior Analysis 📊

## Overview

Customer Behavior Analysis is a data analytics project focused on understanding customer purchasing patterns, preferences, and business insights using data-driven techniques.

The project involves loading and analyzing customer data using Python, performing Exploratory Data Analysis (EDA), cleaning and preprocessing the dataset, executing SQL queries using PostgreSQL/MySQL, and creating an interactive Power BI dashboard to visualize important business metrics.

The main objective of this project is to transform raw customer data into meaningful insights that help understand customer behavior, improve decision-making, and identify key trends.

---

## Dataset

The dataset contains customer-related information used for analysis, including:

- Customer demographics
- Purchase details
- Product information
- Transaction data
- Spending behavior
- Customer preferences

The dataset was analyzed to identify patterns, trends, and relationships between different customer attributes.

## Project Workflow / Steps

### 1. Data Loading

- Imported the dataset into Python environment.
- Loaded data using Pandas for further analysis.

### 2. Exploratory Data Analysis (EDA)

Performed EDA to understand the dataset:

- Checked dataset structure
- Identified missing values
- Analyzed statistical information
- Studied customer trends
- Created visualizations for better understanding

### 3. Data Cleaning & Preprocessing

The dataset was cleaned by:

- Handling missing values
- Removing duplicate records
- Correcting data formats
- Preparing data for analysis

### 4. SQL Analysis

The cleaned dataset was imported into PostgreSQL/MySQL.

SQL queries were performed to extract insights such as:

- Customer purchase patterns
- Revenue analysis
- Customer segmentation
- Product performance
- Business trends

### 5. Power BI Dashboard

An interactive dashboard was created using Power BI to visualize:

- Total customers
- Revenue insights
- Purchase behavior
- Customer trends
- Key performance indicators (KPIs)

## How to Run

### Step 1: Clone the Repository

```bash
git clone https://github.com/PrajwalMudrabett/Customer_Behavior_Analysis.git
```

### Step 2: Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### Step 3: Run Python Analysis

Open the Python/Jupyter Notebook files and execute the cells to perform:

- Data loading
- EDA
- Data cleaning
- Visualization

### Step 4: Database Analysis

- Import the cleaned dataset into PostgreSQL/MySQL
- Run the SQL queries provided in the project folder

### Step 5: Open Power BI Dashboard

- Open the `.pbix` file
- Refresh the dataset connection
- Explore the dashboard
